# 项目简介 java1013

本项目是一款基于SpringBoot和Vue的社团管理系统，集成了多种功能模块，旨在帮助社团高效便捷地进行日常管理和活动组织。

## 技术栈

### 后端

- SpringBoot

### 前端

- ElementUI
- Vue

## 功能模块

系统主要包括以下功能模块：

1. 用户管理
2. 社团管理
3. 活动管理
4. 资源管理
5. 通知公告

## 系统角色

系统定义了以下角色：

1. 管理员
2. 社团负责人
3. 普通成员

## 运行环境

- JDK 1.8+
- MySQL 5.7+
- Node.js 12.0+
